# 小说阅读系统项目完成总结报告

## 项目概述

小说阅读系统是一个全栈Web应用，采用前后端分离架构，提供用户认证、小说管理、阅读、搜索、评论评分、审核等完整功能。系统支持多种格式的小说上传与阅读，具备完善的社交功能和管理员审核机制。

## 技术架构

### 后端技术栈
- **语言**: Go 1.24.0+
- **Web框架**: Gin v1.11.0
- **数据库**: MySQL 5.7+ (GORM v1.31.1)
- **缓存**: Redis (go-redis/v8 v8.11.5)
- **认证**: JWT (golang-jwt/jwt/v4 v4.5.2)
- **全文搜索**: bleve库 v2.5.7

### 前端技术栈
- **框架**: Vue.js 3 (3.4.0+)
- **UI库**: Element Plus 2.4.2
- **状态管理**: Pinia 2.1.7
- **构建工具**: Vite 4.5.0
- **EPUB阅读器**: epubjs

```bash

go run main.go

# 启动前端
cd xiaoshuo-frontend
```
